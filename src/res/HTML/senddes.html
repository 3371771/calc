Отправка файла

Для того чтобы обмениваться зашифрованными файлами нужно:

 

1. Зашифровать файл

алгоритм DES

 

2. Отправить другу/коллеге ключ, он генерируется автоматически и лежит в папке рядом с приложенеим. Называется secKeyDES. И еcли у него нет самого приложения, то и его тоже.

 

3. Для корректоной работы ключ должен находиться в одной папке с приложенеим.

 

4. ВНИМАНИЕ! Для зашифровки и расшифровки должен быть использован один и тот же ключ!

 

5. Имя ключа изменять нельзя. Вообще можно, но тогда работать не будет)

 

6. При каждой новой зашифровке генерируется новый ключ. Это повышает безопасность обмена информацией.
